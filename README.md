# Binance ICP-USDT Fee Calculator

## 功能
计算指定时间范围内 ICP-USDT 现货交易的手续费，并计算返佣金额。

## 使用方法

```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python src/main.py --api-key YOUR_API_KEY --api-secret YOUR_SECRET --start 2024-01-01 --end 2024-12-31
```

## 配置

可以通过以下方式配置 API：
1. 命令行参数
2. 环境变量
3. 配置文件 (config.yaml)

详见 README.md
